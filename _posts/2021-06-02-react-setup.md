## Installation

`npm install react`

## Create a React App
`create-react-app .`

## Importing React
`import React from 'react';`
`import ReactDOM from 'react-dom';`

## Create the App.jsx File

```
import React from 'react';

const App = () => {
  return <div>App</div>
};

export default App;
```

## Create index.js

```
import React from 'react';

reactDOM.render(<App/>,d document.getElementByID("root"));

export default App;
```
